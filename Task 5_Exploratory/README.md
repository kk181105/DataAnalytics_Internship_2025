# Task 5 – Exploratory Data Analysis (EDA)

**This task focuses on performing Exploratory Data Analysis on the Titanic dataset using Python.**

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Files Included
- titanic_dataset.csv – Dataset used for analysis
- EDA_Titanic.ipynb – Jupyter Notebook with EDA
- EDA_Report.pdf – Summary of insights
- README.md – Documentation

## Key Insights
- Females had higher survival rates than males
- Higher fares correlated with higher survival
- Passenger class affected survival probability

## Outcome
This task demonstrates the ability to analyze data visually and statistically to extract meaningful insights.
